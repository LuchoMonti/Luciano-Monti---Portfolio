Análisis Exploratorio de Reseñas de Vinos (Python)

Este proyecto se centra en el Análisis Exploratorio de Datos (EDA) para identificar patrones y detectar anomalías en las preferencias de los consumidores.

Objetivo: Transformar datos desestructurados en información estratégica para el sector retail.

Herramientas: Utilización de lenguajes de programación como Python y sus librerías para la automatización de tareas de limpieza.

Competencias: Demostración de pensamiento crítico y formulación de hipótesis basadas en datos históricos.
